SEZ 3 I PRINCIPI DEL WEB DESGIN

I 6 elementi generali del web design
elementi che si ritrovano in tutte le pagine web
in una pagina web troviamo:
- linee
- forme 
- colori
- background
- tipografia (tipo carattere)
- immagini e icone

hanno scopo rendere comunicazione gradevole pagina web in senso estetico
e efficacie ed efficiente

discrezione
il web design moderno è molto essenziale e disceto, 
nessuno degli elementi è mai invadente anche quando gli oggetti sono animati
cioè gli elementi non si vanno a 


LINEE
una linea che crea una separazione tra una sezione e un'altra (il colore background è diverso )
separare la diversità dei contenuti
elementi/sezioni con contenuti diversità

FORME
sono box, contenitori: sono semplici ed essenziali

COLORE
il colore  mette in evidenza, mette il focus su un oggetto un testo
campitura a colore pieno e anche con quanta armonia viene creata
(es iphone color rosso con background rosso)

box diversi con colori diversi
il colore espremi un certo sentimento, è un portare di simbolo


BACKGROUND
non deve distrarre
banditi background con colorazione eccessiva, troppo figurativa
si può separare i box tra colori diversi
si può realizzare anche con una tessitura

TIPOGRAFIA
scelta di un certo tipo di font rispetto ad un altro
grandezza, colore, come allineato (centratro dx), spaziatura all'interno

IMMAGINI/ICONE
elemento essenziale

13) Anatomia di una pagina web: i principi del web design
architettura funzionale all'usabilità dell'utente
friendly: facilità come l'utente trova una informazione
comunicativo: la comunicazione deve essere efficacie

- Bilanciamento
	cromatico: sfondo bianco/colori neri oppure viceversa
	contrasto cromatico 
	forte dissonanza bianco nero
	anche la disposizione delle immagini (es formano un triangolo)
	
- Ritmo: ti da il senso del movimento ed ha in se per se il senso della ripetizione
	è dato dal ripetere la stessa forma della pagina,
	es: stesso tipo di contenitore (rettangolo senza smussatura degli angoli,
	   immagini interne che seguono la stessa angolazione)
	   

- Proporzione: utilizziamo sia a livello tipografico (titolo grande, titolo più piccolor
	titolo più grande: mette in evidenza un messaggio
	proporzione delle immagini e delle dimensioni
devono dare l'idea di come sia nella realtà: es diversi telefoni modelli diversi proporzianati nella dimensione
immagine enorme su un campo bianco

- Spazio
	l'utilizzo universale delle grandi pagine bianche, o cmq chiaro,
	sfumature del grigio (+/- tenue).

- Dominanza
	la comunicazione è legata al marketing
	mettere in evidenza dominante background e oggetto sopra il background ritagliata
	campitura di colore (background rosso e oggetto rosso, iphone rosso)

- Gerarchia
	osservare un principio gerarchico:
	presenza immagini dominanti, che rimangono impresse, e
	vengono messe anche delle scritte in evidenza insieme al titolo
	
	le sezioni di divisioni devono avere importanza diversa, gerarchicamente offrono importanza diversa


- Unità o uniformità (pagina uguale in tutti gli aspetti, ci sono anche degli aspetti di rottura, ma voluti: focalizzare l'attenzione da un oggetto all'altro)
	stile che una pagina web deve avere
	usare lo stesso font per tutti i titoli delle pagine, sottotitoli
	
	sezioni: titolo stessa posizione e allineamento + fotografia
	deve essere uniformità dello stile 
	stesse icone per mettere in evidenza parti della pagina le stesse
	
palette di colori: scelgo pochi colori da utilizzare

RUOTA DEI COLORI: cerchio di colori:
	

bianco: insieme sette colori arcobaleno

colori neutri: bianco e nero


colori primari, puri che non nascono dal nulla: rosso blu giallo
	
secondari: mescolanza di due primari
	verde: giallo 50% + blu 50%
	viola: 50 blu e 50 rosso
	arancione: 50 giallo + 50 rosso
	grigio: 50 nero + 50 bianco
	
terziari: mescolanza di un primario con un secondario

complementari: sono opposti nella ruota dei colori

analoghi: seguono i primari nella ruota

monoscromatici: in scala rispetto alle ombre e alla tonalità

temperature dei colori:
	colori caldi
		giallo, rosso, arancione
		
	colori freddi
		blu, verde, viola

	colori neutri
		nero, bianco, grigio

nero:  negazione di tutti i colori


tonalità espresa dalla hue: tinta o tonalità dei colori
	quanto colore c'è
	
	saturazione: quantità di colore
			carico di quel colore
			
brightness/lithness: quantità di luce

cosa è HTML
E' un linguaggio di marcatura letto nel browser che indica allo stesso la struttura
che dovrà essere visualizzata per l'utente
Quindi, indica che tipologia di elementi utilizza quella specifica pagina wev
al di là dello stile che essi assumeranno

lo standard attuale è HTML5

è errato definire HTML un linguaggio di programmazione

è lo scheletro di qualunque web app

dice al browser questa è un'immagine, un titolo, etc




Cosa è (...sono...) CSS, è il come sono
Cascading Style Sheet
 
Anch'essi letti dal browser e secondo le regole impostate ne visualizza il rendering. I fogli di stile dicono che aspetto avrà quel particolare tag HTML.
 
Lo standard attuale è CSS3
E' errato definire CSS un linguaggio di programmazione

HTML e CSS lavorano insieme
HTML costruisce la struttura
CSS da lo stile allo scheletro --> a cascata: c'è una regola di ereditarietà delle proprietà

Cosa sono HTML e CSS
In conclusione, il browser legge il codice HTML collegato ad un foglio di stile e quando si trova davanti ad un tag HTML si chiede: "Tu chi sei?"

Il tag HTML risponde: "Sono un titolo di un certo livello".

Bene.

Ma non basta e il browser chiede ancora: "Ma come sei fatto e dove sei posizionato esattamente?"

Il tag HTML risponde: "Non so esattamente ma puoi chiedere al CSS che è collegato a me!"

Il CSS risponde inviando le sue specifiche: "Quel titolo deve essere rosso, grande tanto, allineato al centro e distante tanto dalla riga seguente. " A questo punto il browser ha tutte le informazioni per visualizzare quello che il web designer voleva.

 

Cosa è SVG
Scalable Vector Graphics
 

Quando lavoriamo con grafica vettoriale, utilizziamo grafica sempre perfettamente scalabile a differenza della grafica raster che perde dettagli o risulta "pixellata" quando ridimensionata.

SVG invece utilizza vettori che, ridimensionati, non perdono qualità.

 

I file SVG sono di fatto dei semplici file di testo, sono oltretutto molto leggeri e facili da editare.

 

Particolarmente indicato questo formato per la corretta visualizzazione nel disegno reattivo delle interfacce web
